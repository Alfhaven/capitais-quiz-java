# Capitais Quiz

## Status do projeto
Em desenvolvimento

## Tecnologias utilizadas
- Java SE
- Java Swing
- MySQL
- JDBC
- MySQL Workbench
- NetBeans IDE

## Time de desenvolvedores
- Alvaro Santos

## Objetivo do software
O sistema tem como objetivo oferecer um quiz educacional sobre capitais
federais e estaduais, permitindo o cadastro e gerenciamento de dados
geográficos e o registro de resultados dos usuários.

## Funcionalidades do sistema
- Cadastro de continentes
- Cadastro de países
- Cadastro de estados
- Cadastro de capitais (federais e estaduais)
- Painel administrativo
- Execução do quiz
- Registro de resultados
